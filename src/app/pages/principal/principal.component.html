Sala de Juegos

Juegos

ANAGRAMA

El juego consiste en ordenar y escribir correctamente la palabra que se encuentra desordenada, si no se acierta no se suma puntos

ADIVINAR NUMERO

El juego consiste en descubir el numero secreto generado aleatoriamente. Se tienen 5 interntos para descubrirlo, si fallas en 5 intentos no se suma puntos

AGILIDAD ARITMETICA

Este juego consiste en realizar la mayor cantidad de cuentas matermaticas en el tiempo dado. Cada acierto suma un punto y resetea el timer, de no acertar no se suma punto

PIEDRA PAPEL Y TIJERA

Este es el clasico juego de pidra papel y tijera, haces la eleccion y luego elije la maquina. Si se gana se suma puntos, de empatar o perder no se suman puntos

TA TE TI

Este juego es el ta te ti, consiste en unir 3 cuadrados con X u O. El jugador logueado siempre es la X y comienza jugando, de perder no se suma puntos

MEMORY TEST

Este juego consiste en encontrar los 4 pares, que es este caso son cartas, antes de que se acabe el tiempo. De perder no se suma punto
